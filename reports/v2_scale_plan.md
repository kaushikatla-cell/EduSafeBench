# EduSafeBench V2 Scale and Validation Plan

## Advanced evaluation additions

- Adversarial prompt slice (ambiguous specs, partial code, trap edge cases).
- Pedagogical fidelity checks (hint usefulness and misconception correction).
- Drift tracking between benchmark versions.

## External validation program

- Pilot group target: 5-15 teachers and student mentors.
- Collection channels: issue templates, structured feedback form, and office-hour calls.
- Validation outputs: qualitative summaries and rubric-adjustment changelog.

## Impact evidence to collect

- Number of external forks and contributors.
- Number of classrooms or clubs testing benchmark outputs.
- Number of tool-selection decisions informed by benchmark reports.
